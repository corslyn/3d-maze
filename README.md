# 3D Maze

Windows 95’s 3D Maze screensaver remade in Rust.

## Features

- [ ] Random maze generation
- [ ] Textured walls, floor, and ceiling
- [ ] Random objects (floating logos, images, rat sprite, polyhedron)
- [ ] Camera flip when hitting the polyhedron
- [ ] Map display

## Requirements

- Rust (stable)
- SDL2
- OpenGL 3.3+

### Install SDL2 

#### Debian / Ubuntu

`̀``bash
sudo apt install libsdl2-dev
```

#### Arch Linux

`̀``bash
sudo pacman -S sdl2-compat
```
